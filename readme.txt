README 
---------------------------------------------------------------------------------------

This folder contains R script to generate visualisations for the study:
Vaccine coverage and equity impact in Ethiopia

plot.R - plot script that generates
    (i) plot_socioeconomic_geographic_maternal_child_coverage.jpg (and .eps figures)
   (ii) plot_aor.jpg (and .eps figures)


ethiopia_map.R - plot script that generates
  (Iii) Ethiopia_vaccination_coverage_DHS2016.jpg (and .eps figures)

---------------------------------------------------------------------------------------

data files 

    (i) data_coverage.csv
   (ii) data_aor.csv
  (iii) eth_coverage.csv

---------------------------------------------------------------------------------------

    (i) plot_coverage_maternal_child.png
